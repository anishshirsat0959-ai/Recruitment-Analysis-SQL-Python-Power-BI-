# Recruitment-Analysis-SQL-Python-Power-BI-
Interactive Power BI dashboard analyzing India’s 2026 job market trends, skill demand, hiring patterns, company openings, and work mode insights using SQL and Excel.
🚀 Project Showcase: Recruitment Analytics Dashboard (SQL + Python + Power BI)

Excited to share an end-to-end data analytics project where I used SQL, Python, and Power BI to analyze recruitment data and generate meaningful business insights.

🔹 SQL – Data creation and analysis
🔹 Python – Dataset generation and CSV export
🔹 Power BI – Interactive dashboard creation

📊 Key Features:
✅ Total Applications Tracking
✅ Status-wise Analysis (Selected, Rejected, Pending)
✅ Company-wise Application Distribution
✅ Interactive Filtering using Slicers

This project helped me gain hands-on experience in data analysis, reporting, and dashboard development while working with multiple technologies in a single workflow.
